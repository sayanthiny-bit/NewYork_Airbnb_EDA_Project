# Exploratory Data Analysis of NYC Airbnb Listings (2024)

This project performs Exploratory Data Analysis (EDA) on Airbnb listings in New York City (2024) to uncover insights into pricing trends, availability, neighborhood distribution, and host behavior.

---

## 📌 Project Overview

The objective of this project is to analyze Airbnb listing data and extract meaningful insights that can help understand market trends and customer behavior.

Key goals include:
- Understanding the distribution of listings across NYC boroughs
- Analyzing pricing patterns and room type availability
- Identifying relationships between different variables
- Visualizing trends in availability, neighborhood popularity, and host activity
- Preparing and cleaning raw data for analysis

---

## 🛠️ Tools & Technologies

- **Programming Language:** Python  
- **Libraries:**
  - pandas – Data manipulation and analysis  
  - numpy – Numerical computations  
  - matplotlib – Data visualization  
  - seaborn – Statistical visualization  

---

## 📊 Dataset

- **Source:** Inside Airbnb  
- **Dataset:** NYC Airbnb Listings (2024)  
- **Size:** 20,000+ records  

### Key Features:
- `name` – Listing name  
- `host_name` – Host name  
- `neighbourhood` – Location area  
- `room_type` – Type of room  
- `price` – Listing price  
- `availability_365` – Availability throughout the year  

---

## 🧹 Data Cleaning & Preprocessing

- Removed missing and duplicate values  
- Standardized column formats  
- Filtered extreme outliers in pricing  
- Converted data types for analysis  
- Prepared features for visualization  

---

## 📈 Key Insights

- Manhattan has the highest number of listings, especially in high-demand areas like Harlem and Upper West Side  
- Entire homes/apartments dominate the market, followed by private rooms  
- Prices vary significantly depending on location and room type  
- Listings with higher availability tend to have more competitive pricing  
- A small number of hosts manage multiple listings, indicating commercial usage
